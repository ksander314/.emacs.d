* Файлы

| Файл              | Назначение                       | Как попадают задачи     |
|--------------------+----------------------------------+-------------------------|
| inbox.org          | Входящие                         | C-c t                   |
| work.org           | Рабочие задачи по дням           | C-c r / C-c u / C-c C-w |
| projects.org       | Пет-проекты                      | C-c C-w (refile)        |
| interruptions.org  | Лог прерываний (Focus Shield)    | C-c z (автоматически)   |
| reviews.org        | Заметки из код-ревью             | C-c R                   |
| decisions.org      | Журнал технических решений (ADR) | C-c D                   |
| people.org         | Заметки для 1-on-1               | C-c 1                   |
| incidents.org      | Журнал инцидентов                | C-c !                   |
| energy.org         | Трекер энергии                   | C-c E                   |

* Keybindings

** Глобальные (работают отовсюду)

| Клавиши | Действие                           |
|---------+------------------------------------|
| C-c a   | Org agenda (Dashboard: d)          |
| C-c t   | Новая задача → inbox               |
| C-c u   | Срочная задача → today :unplanned: |
| C-c s   | Стендап → work.org + kill-ring     |
| C-c m   | Встреча → work.org + clock in      |
| C-c w   | Weekly Review → kill-ring          |
| C-c z   | Focus Shield: заморозить           |
| C-c Z   | Focus Shield: разморозить          |
| C-c R   | Code Review → reviews.org          |
| C-c D   | Decision Log → decisions.org       |
| C-c 1   | 1-on-1 заметки → people.org        |
| C-c !   | Incident → incidents.org           |
| C-c T   | Timesheet за N дней                |
| C-c E   | Energy check → energy.org          |

** В org-mode (курсор на heading)

| Клавиши   | Действие                          |
|-----------+-----------------------------------|
| C-c i     | → INPROCESS (clock starts)        |
| C-c d     | → DONE (clock stops)              |
| C-c p     | → PAUSE (clock stops)             |
| C-c P     | Pomodoro 25 мин (clock + таймер)  |
| C-c r     | Refile в сегодня (work.org)       |
| C-c l     | Привязать JIRA / GitHub / GitLab  |
| C-c A     | Архивация DONE/CANCELED           |
| C-c C-w   | Refile с выбором (work / projects) |
| C-c C-t   | Выбор статуса: t/s/p/d/c          |
| C-c C-x e | Установить effort (оценка)        |
| C-c C-x C-c | Column view: оценка vs факт    |
| S-↑ / S-↓ | Приоритет A / B / C               |

| Элемент     | Вид                                  |
|-------------+--------------------------------------|
| TODO        | оранжевый (org-warning)              |

| PAUSE       | серый                                |
| DONE        | серый, зачёркнутый                   |
| CANCELED    | зелёный, зачёркнутый                 |

* Типичный день

1. Утро: Emacs открылся → Dashboard с прогресс-баром дня
2. =C-c s= → стендап скопирован, вставил в Slack
3. Открыл inbox → =C-c r= на каждой задаче → они в today
4. =C-c C-x e= на задачах → проставил оценки (0:30, 2:00…)
5. =C-c P= на первой задаче → pomodoro 25 мин → clock + фокус
6. Кто-то написал в Slack → =C-c z= → заморозил контекст
7. Разобрался → =C-c Z= → всё восстановлено
8. Прилетел баг в проде → =C-c != → залогировал инцидент
9. =C-c u= → "Fix crash" → Start now? y → clock переключился
10. Починил → =C-c d= → DONE
11. Code review → выделил проблему → =C-c R= → заметка с кодом
12. Принял архитектурное решение → =C-c D= → ADR в decisions.org
13. 1-on-1 → =C-c 1= → "Иван" → повестка + заметки
14. =C-c E= → залогировал энергию (3/5 после обеда)
15. =C-c i= на следующей задаче → clock вернулся
16. Пятница: =C-c w= → weekly review → скопировал в Slack
17. Вечер: =C-c A= на heading дня → архивация DONE/CANCELED
18. Закрыл Emacs → "Конец дня. Показать итоги?" → да → итоги
